# 🚀 Team Standup Board

A simple full-stack web application where team members can post their daily standup updates — what they did, what they will do next, and any blockers.

---

## 📌 Features

- Add daily updates
- View all team updates
- Clean and simple UI
- Stores data in database
- Filter updates (optional feature)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** MongoDB
